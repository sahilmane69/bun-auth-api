# bun-auth-api

A simple backend authentication API built with **Bun**.

## Features
- User signup & login
- JWT-based authentication
- Password hashing
- Protected routes
- SQLite database

## Setup

Install dependencies:
```bash
bun install
```

Run in development:
```bash
bun dev
```

Run in production:
```bash
bun start
```
